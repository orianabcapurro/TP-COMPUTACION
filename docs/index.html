Ima Pico Generative

Obra visual generativa e interactiva inspirada en la serie "Ima Pico / Q3"

Controles

  • ↑ / + — aumentar intensidad
  • ↓ / - — disminuir intensidad
  • Espacio — cambiar paleta
  • R — mutar una forma al azar
  • X — resetear mutaciones
  • D — mostrar/ocultar debug

Proyecto desarrollado en Processing Java Mode. Esta demo web es un port a p5.js.
